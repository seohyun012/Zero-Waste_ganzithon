### ✨ Key Features
* **적정 처방 안내**: 질병별 평균 약물 처방량 데이터를 제공하여 과잉 처방 방지
* **진료비 비교**: 평균 진료비 및 조제료 정보를 시각화하여 합리적인 의료 이용 지원
* **데이터 기반 분석**: 공공 데이터를 활용한 신뢰도 높은 정보 제공

### 👩‍💻 My Role (Frontend & Design)
* **UI/UX 디자인**: Figma를 활용한 서비스 전체 와이어프레임 및 프로토타입 설계, 전반적인 외형 디자인
